backgroound has 8 different related background properties. border - creates border with size, type and color border-radius is used to create rounded corners on things box-shadow adds shadow effect around element. overflow - defines what happens to element when its content is too big for to fit. if 1 number specified both overflow-x and overflow-y have the same value, if 2 numbers, 1st OF-x, 2nd - OF-y. opacity - allows to make an element more or less transparent. Value range 0.0-1.0 or percentage from 0% to 100%.
