Framework is kind of library with preset styles for everything , so when using it you dont really need to do many manual inputs. Well-known frameworks are Bootstrap and Tailwind. Preprocesors or precompliers are languages that was made to help with quicker code writing. it could help to reduce repetition, custom properties was initially introduced there, joining, multiple css sheets. Frameworks: Bootstrap, Tailwind → ready-made styling systems Preprocessors: Sass, Less → better ways to write CSS
