// The form element Its a container element like the div element we learned earlier in the curriculum, the form element wraps all of the inputs the user will interact with on form. the form takes two attributes: action and method - action is attribute which takes a URL value that tells the form where it should send its data to be processed. - method is attribute which tells browser what HTTP protocol request method it should use to submit the form (GET and POST used the most). GET method used to retrieve information from a server (e.g Google search query); POST is used when we want to chage something on the server, e.g. when a user makes an account or makes a payment on a website.
