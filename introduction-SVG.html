SVG - scallable vector graphics, defined by math. With vector graphics (and absence of grid) we use formulas to scale file to any size or shape we want. SVG defined using XML (Extensible Markup Language) is HTML-like syntax which is used for things like APIs, RSSs, spreadsheets and word editor software. Benefits of SVG being XML are: 1. Human readable, nt like JPEG's binary file format (still confusing tho), 2. Its interoperatable with HTML, without any changes or specifications. Because of that they can be targeted in the DOM like HTML elements.
