Fonts can be: 1. system font stack - system based vanilla fonts 2. web fonts - fonts which downloaded on home machine from web 3. online font libraries - google fonts, font library and etc 4. self-hosted fonts - downloaded and available on machine, using @font-face @font-face { font-family: my-cool-font; src: url(../fonts/the-font-file.woff); } Text styles 1. font-style is used to make text italic/bold/etc. - CSS property to highlight text, however em, strong - used to do the same as font-style , but they have additional purpose to emphasize text. 2 . letter-spacing is used to change the space between leters in a word. Can be used to adjust fonts as well. 3. line-height is usedto adjust space betwee lines in wrapped text. Good to increase readability. 4. text-transform is used to change the case of given text (Upper or Lower or mixed) 5. text-shadow is used to add shadow arouhnd the text in the selected element. 6. elipsis - not a single property, but group of. With text-overflow it is possible to truncate overflowing text with an elipsis. Use font-display: optional for fonts used in body text. line-length can be set by article { max-inline-size: 66ch; } clamp() function - allows locking the font size between sensible minimums and maximums while also providing a scalable middle value.
